This project is a Java Swing application that implements a simple calculator using the MVC (Model–View–Controller) design pattern. The Model is responsible for handling all the mathematical operations such as addition, subtraction, multiplication, and division, while the View provides the graphical user interface where the user can enter values and see results. The Controller acts as an intermediary between the Model and the View by handling user actions, processing input, and updating the output accordingly. This project demonstrates the separation of concerns in software design and can be extended to support binary and scientific calculations such as sin, cos, and square root operations.
